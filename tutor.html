AI Warrior — tutor & reo helper preview

Two widgets powered by one Cloudflare Worker, routed through AI Gateway, running on Workers AI at the edge. Quota-tiered: free for casual visitors, generous for Kiwi Dialectic subscribers, with a Perplexity Pro referral when the meter runs out.

Reo Helper

Translation between English and te reo Māori, macron suggestions, phrasing alternatives. Uses Llama 3.1 8B for speed.

Course Tutor

Socratic discussion of Gramsci, Kropotkin, Graeber, Freire, Deleuze, Bakunin — connected to material conditions. Uses Llama 3.1 70B for nuance.

How this works

Both widgets POST to a Cloudflare Worker (/worker in this repo). The Worker routes the request through your AI Gateway (ai-warrior-gateway) and into Workers AI. Your API keys never touch the browser. Per-IP and per-subscriber monthly quotas are tracked in KV.

The endpoint URL above is substituted at deploy time by the GitHub Pages workflow using the WORKER_ENDPOINT repo variable — see worker/README.md for the full setup.
