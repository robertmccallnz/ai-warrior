# Branding Assets

Long-term visual identity reference files for The Kiwi Dialectic / AI Warrior. Not used directly in `index.html` unless explicitly linked — these are the source-of-truth assets for posters, zines, social, and future site work.

## Files

| File | Notes |
| --- | --- |
| `francis-bacon-portrait.jpg` | Two-colour stencil / screenprint portrait of Francis Bacon — magenta + green on kraft. Patron-saint motif. Use as a section divider, zine cover, or pull-quote backdrop. |
| `musenz-x-avatar.jpg` | Current X/Twitter avatar for [@musenz](https://x.com/musenz). Red koru spiral on black with "KD" wordmark. 400×400 — source asset for the wider Kiwi Dialectic identity. |

## Palette pulled from the Bacon portrait

- Magenta / wine: `#6B1A47`
- Printshop green: `#1F7A4A`
- Kraft paper: `#B8A074`

Use sparingly alongside the primary red/black/white system — this is the "second voice" palette for editorial / long-form / poster work.
